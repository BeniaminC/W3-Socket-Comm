# W3-Socket-Comm

Easy Steps:

1. Install Python 3.12+
2. Run `setup_and_run.sh`
3. Run exe file in dist directory.
4. Change the settings to your likings

    - Default delimiter is a `space`
    - Default hotkey for copying is `CTRL+SHIFT+C`
    - Default path to web UI is `r"C:\Program Files (x86)\Warcraft III\_retail_\webui"`

5. Start it
6. Join a lobby
7. Press hotkey
8. Get battlenet tag names from lobby
